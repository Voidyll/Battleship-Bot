# Web Application Development II Final Project
## Battleship 
This application is recreating the game of Battleship with an AI agent that the user can play against. This will be a web application hosted for only a short time until we pass the class. Will attach the website once it is up and running. Currently working on creating 3 components for this: an AI agent that can play battleship, a RESTful API that controls the gameflow, and a web application frontend for the user to actually play the game of battleship.
