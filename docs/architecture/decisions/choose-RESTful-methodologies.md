# RESTful Methodologies Architecture Decision

## Status

Proposed

## Context

Communication between web client and server.

## Decision

Use JSON/HTTP communication to transfer data from the web client to the server via RESTful methodologies. More info here: https://restfulapi.net/

## Consequences

### Pros
* Data transfer and handling becomes easier 
* Code maintenance become easier

### Cons
* Requires understanding of RESTful methodologies
* Takes extra time to implement properly
